## 주요기능
- 요리하는 사람들을 위한 레시피 공유
- 나만의 냉장고 서비스로 자신의 냉장고에 맞춤 요리 레시피 공유
- 커뮤니티 & 음식 나눔 서비스

## 주요 기술
- MVC 패턴 기반 Spring  RESTful API 서버 구축
- Oracle을 사용하여 DB 설계 및 구축
- AJAX를 이용한 서버 측과 클라이언트측간의 비동기식 데이터 교환 기술
